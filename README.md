# AnimePrayer 🕌
Mengapa namanya Anime Prayer? karena berawal dari set custom foto yang gepeng, meskipun saya buat ini buat nampilin Bias KPOP saya

AnimePrayer adalah aplikasi desktop pengingat waktu sholat kustom untuk Muslim yang dirancang untuk mengingatkan Anda kapan waktunya beribadah. Aplikasi ini dibangun menggunakan Tauri dan React, menawarkan antarmuka modern dengan fitur personalisasi yang kaya.

## ✨ Fitur Utama

* **Jadwal Sholat Akurat**: Menghitung waktu sholat (Fajr, Sunrise, Dhuhr, Asr, Maghrib, Isha) secara otomatis berdasarkan pengaturan koordinat (Latitude & Longitude) dan zona waktu Anda.
* **Hitung Mundur (Countdown)**: Menampilkan timer hitung mundur secara real-time menuju waktu sholat berikutnya secara langsung di dashboard.
* **Notifikasi Kustom**: Anda dapat mengatur teks notifikasi, serta menambahkan gambar dan suara kustom yang akan muncul setiap kali waktu sholat tiba. Anda dapat atur teks, gambar, serta suara sekustom mungkin seperti memasang foto OSHI, WAIFU, KPOP BIAS

* **UI Modern**: Menggunakan desain "Glassmorphism" yang elegan dengan mode gelap (dark theme) untuk kenyamanan mata.

## 📥 Cara Instalasi (Windows) JANGAN MALAS INI GAMPANG TINGGAL INSTAL EXE
![HigurumaJujutsuKaisenGIF](https://github.com/user-attachments/assets/4bd973fa-27b4-4d41-99ec-1954012edd6a)

Aplikasi ini sudah di-build ke dalam bentuk installer (format `.exe`) versi `0.1.1` menggunakan NSIS, jadi kalian tinggal install saja tanpa perlu download  seluruh proyek ini. Untuk menginstalnya, ikuti langkah-langkah berikut:

1. Buka Proyek Github ini.
2. Navigasikan ke direktori build NSIS di dalam folder proyek: 
   `src-tauri/target/release/bundle/nsis/` atau `https://github.com/mifahh/AnimePrayer/tree/main/src-tauri/target/release/bundle/nsis`
3. Cari file installer bernama(pilih versi terbaru) `AnimePrayer_(version)_x64-setup.exe`.
4. Klik Download File, Petunjuk seperti dibawah ini :

![Anime Prayer Tutorial](https://github.com/user-attachments/assets/21610e94-0bcc-47a7-bd75-ea98feb89d80)

5. Setelah file `.exe` ter-download. Klik ganda (double-click) pada file installer tersebut untuk menjalankannya.
6. Jika muncul peringatan keamanan (Windows SmartScreen), pilih **More info** lalu klik **Run anyway**.
7. Ikuti panduan (setup wizard) pada layar hingga proses instalasi selesai.
8. Setelah selesai, Anda dapat mencari dan menjalankan "AnimePrayer" dari Start Menu Windows Anda.
9. Selamat Menikmati
